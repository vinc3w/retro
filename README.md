## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

Follow the steps below to install the project.

1. Clone the repo
   ```sh
   git clone https://github.com/vinc3w/retro.git
   ```
2. Install server NPM packages and run
   ```sh
   cd retro
   npm install
   npm run dev
   ```
